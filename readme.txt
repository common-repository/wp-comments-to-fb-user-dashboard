== Description ==

Give you che possibility to send WP comments (with your custom comments form) to facebook user dashboard. 

It's not providing facebook login (provided from many other plugins) but only the comments connection. Login to fb required with publish_stream permissions. Require CURL.

== Installation ==


1. Unzip and upload it to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure facebook key, secret key and default image in settings

That's it. Enjoy

== Frequently Asked Questions ==

= I can't see comments publishing =

Is your wp user logged in tpo facebook? and autenicated with publish_stream permission?

= Your plugin give the possibility to connect to facebook? =

No, it work with other plugins (there are several) that give you the possibility to login with facebook. It onlu check if autenthicated, and post comments if so
